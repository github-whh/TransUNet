# CKMTransUNet

CKMTransUNet is used to construct beamforming-codebook-aware channel knowledge map of multi-antenna systems.

For more information see the paper: [Beamforming-Codebook-Aware Channel Knowledge Map Construction for Multi-Antenna Systems](https://arxiv.org/abs/2505.16132).

# Model

The pretrained model can be downloaded from [Google Drive](https://drive.google.com/drive/folders/1bJUFdAlnKic8MdM0r8QDP3cmG-SSOkcq).

The trained model can be downloaded: [Google Drive](https://drive.google.com/drive/folders/1bJUFdAlnKic8MdM0r8QDP3cmG-SSOkcq).
