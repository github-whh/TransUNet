# Radio-TransUNet

TransUNet is used to construct beamforming-codebook-aware channel knowledge map of multi-antenna systems

For more informationn see the paper: Beamforming-Codebook-Aware Channel Knowledge Map Construction for Multi-Antenna Systems

# Model
The pretrained model can be downloaded: https://drive.google.com/drive/folders/1bJUFdAlnKic8MdM0r8QDP3cmG-SSOkcq

The trained model can be downloaded: https://drive.google.com/drive/folders/1bJUFdAlnKic8MdM0r8QDP3cmG-SSOkcq

# Dataset

The dataset of beamforming-codebook-aware channel knowledge map of multi-antenna systems can be downloaded: https://drive.google.com/drive/folders/1rXx10-FE3ALH-57TAh9_2ltZEt0JnjDk
